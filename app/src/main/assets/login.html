Log in

When a user accesses the app, it is automatically entered on the login page. The login page assumes two fields: Email and Password.


Next, the user will be redirected to the CLASSBOOK page within the app.


If the user does not have an account already created, you must press the "Sign up!" button, which will redirect it to the page Create Account.


Changing the language

If you want to change the display language of the app, this can be done by pressing the top-right button of the screen and then selecting the language you want.
